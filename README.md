# C Calculator Parser

โปรแกรมสำหรับแยกตัวเลขและเครื่องหมายทางคณิตศาสตร์ออกจากข้อความ (Parsing) เขียนด้วยภาษา C
รองรับทั้งจำนวนเต็มและทศนิยม โดยใช้ Logic การแปลง string เป็น number ด้วยตัวเอง

## โครงสร้างไฟล์
* `calculator.h`: ประกาศโครงสร้างข้อมูล (Struct) และหัวข้อฟังก์ชัน
* `calculator.c`: โค้ดการทำงานหลัก (Logic) ของการแยกตัวเลข
* `main.c`: ไฟล์หลักสำหรับเรียกใช้งานและแสดงผล

## การติดตั้งและใช้งาน

เนื่องจากโปรแกรมมีการใช้ฟังก์ชันทางคณิตศาสตร์ (`pow`) จาก `<math.h>` การคอมไพล์บน Linux/macOS จำเป็นต้อง link library ด้วย flag `-lm`

### คำสั่ง Compile
```bash
gcc main.c calculator.c -o parser -lm
